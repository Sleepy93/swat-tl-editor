# swat-tl-editor
S.W.A.T Target Liberty - Editor 
Platform: PSP
Date: 2007

-----------------------------------------------------------------------------------------------------------------------

Created editor for S.W.A.T Target Liberty PSP game.

# Supported features: 

> LMP (game archive) 

> TEX (texture)

> LUA (scripts)

> PNG (texture)

> TGA (texture)

> MP3 (audio) 

This project contains third party solutions.
